# enDWMvour

## Bootstrap
1. Install the **base system only**
2. git clone this repo
3. run `make bootstrap`
4. reboot
5. run `startx`
6. enjoy

```bash
git clone https://github.com/TruncatedDinosour/enDWMvour
cd enDWMvour
make bootstrap
sudo reboot
startx
```
